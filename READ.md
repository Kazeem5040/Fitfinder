
# FitFinder

FitFinder is a web application that helps users discover nearby gyms and generate personalized workout plans based on their fitness goals, experience level, available workout time, and target muscle groups.

The goal of this project is to make getting started with fitness easier by helping users find gyms near them and receive customized workout recommendations in one place.

## Features

- User registration and login system
- Search for nearby gyms using location and distance preferences
- Save favorite gyms for future reference
- Generate personalized workout plans
- Customize workouts based on:
  - Fitness goals
  - Energy level
  - Available workout time
  - Target body parts
  - Height and weight
  - Gym experience level
- View workout history
- Secure user accounts and saved data

## Installation

1. Clone the repository

bash git clone https://github.com/Kazeem5040/Fitfinder.git 

2. Navigate into the project directory

bash cd Fitfinder 

3. Create and activate a virtual environment

bash python -m venv .venv 

4. Install dependencies

bash pip install -r requirements.txt 

5. Create a .env file and add your API keys

env GOOGLE_API_KEY=your_key_here OPENAI_API_KEY=your_key_here 

6. Run the application

bash python run.py
